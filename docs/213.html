结论

PyTorch自定义数据集类是一个强大的工具，可以为非完美结构化的数据创建管道。我们在 getitem 部分使用了自己的翻译词典。我们能够创建和加载我们自己的定制培训标签。甚至有更多的可能性。我们可以创建修改后的文件列表(例如，只过滤特定的品牌)并将它们添加为条件。我们也可以在 getitem 部分中集成 if else 语句。如果您希望将数据集类用于生产中的未标记数据，这将非常有用。例如，通过添加一个 test == true 语句，我们可以返回一个没有标签的字典。PyTorch自定义数据集类提供了无限的可能性来单独构建您的训练数据。感谢阅读！
