# 介绍

正如标题所暗示的，初级和高级数据科学家之间有很多明显的差异，但有哪些鲜为人知的差异呢？在本文中，我们将讨论这些差异，以及一些高级数据科学家代替初级数据科学家可能需要执行的一些关键职责或流程。

首先，需要注意的是，并不是每个公司都有初级、普通和高级员工，所以这种比较只在他们有初级、普通和高级员工的情况下有效。然而，经验丰富的人与经验不足的人之间的比较可以遵循相同的方向。话虽如此，下面让我们更深入地探讨这两个角色。

> 注意:

了解这两个职位的差异不仅有助于*自己*决定哪一级别的职位适合你，也有助于*招聘人员*和*经理*招聘不同资历的员工。有了这些信息，你就可以知道作为一名学生应该学习哪些课程(例如，你会更专注于编程，还是会更多地参与商业和交流？).另一方面，经理和招聘人员可以利用这些信息来设计和撰写职位描述。此外，这些区别也有助于其他*利益相关者(*例如，软件工程师、数据工程师、业务分析师和产品经理*)，*他们可能与这些不同级别的数据科学资历一起工作，因此可以在工作场所设定适当的期望。

# 产品经理协作

![](img/05d0507aaea879a4258d60ab7258c127.png)

照片由 [ThisisEngineering RAEng](https://unsplash.com/@thisisengineering?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 在[Unsplash](https://unsplash.com/s/photos/sticky-notes?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText)【2】上拍摄。

正如所料，初级数据科学家的思维倾向于跟随围绕机器学习算法的学术界，而高级数据科学家的思维则专注于业务影响和协作。

> 例如，下面是初级与高级数据科学家的典型日/周示例:

*   **初级数据科学家** —

—接受来自数据科学经理的任务，如“清理”或预处理数据，以及比较算法

—使用 SQL 查询数据库表中的数据

—根据现有数据创建新要素

—为模型准确性创建可视化效果

以上只是作为初级或普通级别的数据科学家，您每天可能会遇到的一些步骤。你可能会专注于任务本身，而不是为他人创建和组织任务。

*   **高级数据科学家** —

—与产品经理、其他利益相关方合作，制定您自己的任务、路线图和流程—讨论重要的 KPI(*关键绩效指标*)，这些 KPI 在模型准确性和误差本身之外推动模型的开发。例如，不只是优化 MAE，而是优化中间分位数两侧的预测。

在某些公司，这个角色可能更类似于数据科学经理，而在其他公司，它是一个高级数据科学家职位，不管理任何人，而是管理项目本身。这也取决于你周围的团队。

> 您的合作伙伴和业务:

现在，我们对这些角色的日常任务有了更好的理解/只是一些例子，我们现在可以看到高级数据科学家更多地与产品经理合作。不是每个和你一起工作的人都熟悉梅伊、MAPE、RMSE、调整参数、过拟合等。企业最关心的是业务问题的最佳模型，这是您作为一名高级数据科学家需要阐明的事情，而初级或普通级别的数据科学家往往只关注降低错误本身。然而，最有可能的是，有些公司的这些任务是重叠的。

*总结一下*，学长更多的是与人合作，而学妹可以更低头。同样，这只是我收集的经验，所以这在不同的公司会有所不同。

# 项目生命周期管理

![](img/842ca52116258ed2d8418727feb1c100.png)

照片由 [Erlend Ekseth](https://unsplash.com/@er1end?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 在[Unsplash](https://unsplash.com/s/photos/circle?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText)【3】上拍摄。

与上述观点类似，高级数据科学家可能会从头到尾管理更多的项目，而初级数据科学家可能会专注于整个过程的前半部分或中半部分，例如。

> 初级数据科学家项目示例:

*   数据集的 SQL 数据库、预处理数据、特征工程、模型比较

> 高级数据科学家项目示例:

*   收集利益相关者的需求，问题是什么，为什么数据科学是一个可能的解决方案？从 API/SQL 获取数据，预处理数据，特征工程，模型比较，部署模型-可能与机器学习运营工程师或软件工程师合作或自己动手，自动化任务，自动化指标，报告指标，讨论模型如何帮助客户，节省资金，节省时间等。

如你所见，有一个比模型本身更大的生命周期，围绕着它有整个过程。初级数据科学家专注于模型，而高级数据科学家可以专注于模型以及我们为什么首先需要模型。
