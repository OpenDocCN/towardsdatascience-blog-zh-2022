# 什么是训练有素的模特？

> 原文：<https://towardsdatascience.com/what-is-a-trained-model-5c872cfa8448>

## 或者……什么叫“训练一个 ML 模型”？

![](img/dca74b54950d6de4dcf6a00bd3de2c9f.png)

杰森·古德曼在 [Unsplash](https://unsplash.com/s/photos/training?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上的照片

W 当谈到机器学习时，我们总是谈论“训练模型”或“训练模型”,但是……那到底是什么意思呢？

什么是训练有素的模特？我们训练模特的时候都做些什么？我们总是要训练一个模特吗？

在本文中，我们将回答所有这些问题，讨论监督和非监督学习，我们将了解训练机器学习模型在何种意义上不同于训练人类。

# 1.训练一个人工智能模型不同于训练一个人(或动物)

如果我们在字典中搜索动词“训练”,我们会找到这样的内容:

> 通过一段时间的练习和指导教会(人或动物)一种特殊的技能或行为

但是还有更重要的事情。事实上，我更喜欢:

> 教导以使适合、合格或熟练
> 
> 通过指导、训练或操练来形成
> 
> 为技能测试做准备(如通过练习)

这三个有“隐藏”的含义:当一个人(或动物)被训练时，他/她/它在某一点上变得独立。我的意思是，受过训练的人/动物在某一点上可以独立完成训练的任务(没有训练员)。

例如，如果你曾经尝试过任何运动，你就知道它是如何运作的:在开始时，教练会在第一个动作中帮助你，甚至在身体上帮助你。然后，在某一点上(当你被训练的时候！)你自己去吧！

那么，我们如何训练计算机呢？

训练一个 ML 模型并不意味着拿一个机器人，让它骑上自行车，训练它循环移动它的腿并告诉它“上下移动你的腿，你就可以走了！”。这就是我们对人类所做的，但人类(和动物)与计算机有着天壤之别。

因此，在训练 ML 模型时，我们必须将讨论分成两部分:

*   监督学习
*   无监督学习

让我们来看看细节。

# 2.训练一个 ML 模型:监督学习

监督学习(SL)旨在作为人类训练 ML 模型的方式。这意味着人类(数据科学家、机器学习工程师或其他任何人)在一组已知数据中训练 ML 模型。换句话说，引用维基百科:

> **监督学习**是基于示例输入-输出对学习将输入映射到输出的函数的机器学习任务

因此，在**监督学习中，**我们有一组数据，其中**我们知道输入和输出**(我们也说:**数据被标记为** ) ，我们训练 ML 模型来预测看不见的数据的输出。

我们举个简单的例子。考虑一个简单的分类问题，比如我做的[这个简单的](/ive-trained-my-first-machine-learning-model-and-i-got-sad-8abdc0c5b72b)项目。这里我有一个与糖尿病相关的数据集:输入是患者的各种医疗状况，输出是:患者有糖尿病(1)患者没有糖尿病(0)。这意味着一个重要的事实:有人测量了患者的状况，并将其存储在某个地方，还报告了他们是否患有糖尿病。**在这种情况下，**ML 模型的目的是在给定一些输入条件**(他/她可能有的医疗条件)的情况下，预测患者是否可能患有糖尿病。**

在这种情况下，我训练了两个不同的 ML 模型(K-nearest neighborhood 和 Decision Three ),以决定哪一个对看不见的数据提供最好的性能。但是如果我们有一个已知数据的数据集，我们如何处理未知数据呢？在有监督的情况下训练一个 ML 的过程非常简单:

*   我们得到了数据集，知道了输入和输出
*   我们将数据集分成两个数据集:训练数据集和测试数据集
*   我们在训练数据集上训练 ML 模型
*   我们在测试数据上评估了 ML 模型的性能。

因此，测试数据是看不见的数据(对于我们的 ML 模型是“看不见的”)，我们可以评估我们的 ML 模型的性能，并选择我们认为最好的一个。

因此，在监督学习中，**你明确地告诉 ML 模型它应该为每个输入产生**的输出，如果它没有产生输出，那么**你调整它的(超)参数**，以便下一次它更有可能为该输入产生正确的输出。

这是在监督学习案例 : **中训练 ML 的**本质:调整(超)参数**(也称为“超参数调整”)，以便 ML 模型产生尽可能接近期望输出的输出(这是已知的！).**

# 3.训练一个 ML 模型:无监督学习

在**无监督学习** (UL)案例中**数据没有被标记**。如果我们考虑之前的案例，我们有一个病人健康状况的数据集，我们知道他们是否患有糖尿病，在无监督学习的情况下，我们只有病人的医疗状况。句号。我们对他们(可能的)糖尿病状况一无所知。

那么，在这种情况下，培训是如何进行的呢？

无监督学习使用机器学习算法来分析和聚类(意思是“分组”)未标记的数据集。**这些算法在不需要人为干预的情况下发现数据中隐藏的模式**，这也是案件“无人监管”的原因。因此，无监督学习模型可以自己发现未标记数据的“固有”结构。这意味着一件简单的事情:**没有关于无监督学习案例**的训练。在这种情况下，该算法自行寻找如何对(未标记的)数据进行分组，因此人类不训练该模型。

# 结论

在本文中，我们学习了机器学习中的一个重要概念；这就是我们所说的**训练**。训练 ML 模型意味着屏幕后面的人调整模型的超参数，使得模型可以预测尽可能接近真实输出的输出。这意味着，作为人类(训练者)，我们必须知道实际的输出，这就是为什么我们只在我们所谓的“监督学习”中训练 ML 模型。

在无监督学习中，ML 模型自行学习，不需要训练。当然，在可能的情况下，评估我们的无监督模型所做的事情总是一个好主意；为了做到这一点，我们在已知数据上训练一个监督的 ML 模型(但是，正如我之前所说的，这并不总是可能的…否则，就不需要无监督的模型了！).
