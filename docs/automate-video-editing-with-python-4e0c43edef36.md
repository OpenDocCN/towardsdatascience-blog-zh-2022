## 目标

您想要剪切视频，删除背景音乐，并创建带有音频评论的短片。

1.  加载原始视频
2.  移除背景音乐
3.  将视频剪切成几个镜头，分别聚焦于不同的过程
4.  添加通过文本到语音转换解决方案生成的音频注释
